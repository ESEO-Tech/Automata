This is an automatic gate opener with a call button. Program the system to achieve the following behavior:

  1. When the car driver pushes the button, open the gate.
  2. When the gate is fully open, close it after 3 seconds if no vehicle moves through the gate.
  3. When the vehicle has moved past the gate, close it.
  4. If another driver pushes the button, or if a car attempts to move through the gate while it is closing, open it again.

In this setup, we add a counter that can measure a duration of 3 seconds.

Tip: Use the vehicle sensor to detect whether a car is passing through the gate.

Sensors

B
(Button) 		1 The button is pressed
0 The button is released
O
(gate is Open) 		1 The gate is completely open
0 The gate is not completely open
C
(gate is Closed) 		1 The gate is completely closed
0 The gate is not completely closed
V
(Vehicle) 		1 A vehicle is passing through the gate
0 No vehicle is passing through the gate
TO
(Timeout)) 		1 The timer has finished counting 3 seconds
0 The timer has not finished counting 3 seconds

Actuators

U
(Up) 		1 Move the gate up
0 No effect
D
(Down) 		1 Move the gate down
0 No effect
TR
(Timer Reset) 		1 Reset the timer to its initial value
0 Do not reset the timer
TE
(Timer Enable) 		1 Increment the timer
0 Do not increment the timer

Setting both Up and Down at the same time can damage the gate. It is considered as an error.

It Timer enable and Timer reset are enabled at the same time, the Timer reset action is applied.
